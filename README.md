# Dados Pessoas

Tem-se um conjunto de dados contendo a altura e o gênero (M, F) de N pessoas. Fazer um programa <br>
que calcule e escreva a maior e a menor altura do grupo, a média de altura das mulheres, e o número <br>
de homens. <br>
